# 第076章 露雨来取未交接

凉生没有碰回执。

他先让所有人离开照片墙两米，连杜岚也没靠近。身份修复部旧章可以证明某次转诊曾被接收，却也可能把触碰者写成未完成交接的下一任经手人。

露雨说她来取，不等于她已经具备安全取走的权限。

水痕在玻璃上停了几秒，又补出一句：这次不是提示。

“看得出来。”凉生说，“提示一般不负责搬东西。”

侧门外的晨光暗了一下。露雨从消防栓投下的影子里走出来，右手几乎透明，只剩腕骨附近一圈模糊轮廓。她左手提着一只旧档案袋，袋口盖有撤销部门的作废章。

杜岚第一次清楚看见她，目光从脸移到右手：“你是旧部门的人？”

“我来接一份旧部门材料。”

“我问的是你是谁。”

“现在回答，会让这张回执把我的身份补进七年前。”露雨说，“你可以不信，也可以阻止我拿。”

她没有要求凉生替自己背书。

杜岚走到杜驰身边，先看回执上的转诊编号。编号末两位与七张推床的蓝章批次相同，但日期栏被折住。她只确认自己知道的：“这类袋子以前装身份复核转诊，不装完整病历。取走后，医院会少一份留存。”

“所以我带了接收副本。”露雨举起左手的档案袋，“原件不离开现场，只把未交接状态转为外部保全。”

周启明从后面赶来，看到露雨时神色没有明显变化，终端却自动进入离线。他说：“撤销部门无权接收现行材料。”

露雨反问：“你承认这是现行材料？”

周启明被自己的话卡住。若是历史废件，他没有阻止权限；若是现行材料，旧楼便仍有待接服务。

他改口：“来源未核。”

“可以。”露雨说，“那就在公开监控下核，不用你的部门给我结论。”

她把档案袋放在地上，没有靠近相框。袋内只有三样东西：旧部门材料接收清单、跨部门转诊骑缝章样本、以及一张写有当前日期的责任声明。

声明由她本人签名，只承担这次复制、封存和送达，不承担七年前原签名、空相框身份或患者000死亡结论。

凉生看完，没替她补充。他问：“代价呢？”

“右手会失去一段现实接触。”

“多久？”

“不确定。”

“还取？”

露雨看向相框后的回执：“我追这条线，不是因为你在这儿。旧部门被撤销后，所有未交接材料都被算成从未发出。拿不到这份，七条记录明天会再次没有送达对象。”

她的目标和凉生此刻一致，却不是围着他产生的。

杜岚问：“能帮我查N-204和杜驰吗？”

“能保留查询，不保证给答案。”

“会不会拿他们补你的部门记录？”

“不会。你可以看完整个操作，也可以随时叫停。”

杜岚盯着她看了很久，最终点头：“我看。”

露雨开始取件。

她先用左手把接收清单贴在相框外，右手穿过裂缝。透明的手指接触回执时，玻璃里突然伸出七八只戴手套的手，抓住她手腕。每只手套上都是不同年份的值班日期。

未交接材料把所有过去经手人压在同一个取件动作上。

露雨没有喊。她咬住档案袋封绳，用左手逐年撕开叠加日期。可撕到第三张，右臂已经淡到肘部。

凉生下意识往前一步。

“别接。”露雨声音发抖，“你接，它会把你写成七年前那一笔。”

他停住，没有用救她的冲动毁掉她设下的边界。转而问杜岚：“这些手在执行什么？”

“交班留置。材料没被下一岗签收，上一岗不能松手。”

“那就让今天的接收成立，不用替换上一岗。”

凉生把露雨当前责任声明提交给施工现场的公开见证，不走医院内部交班。蒋师傅和保安大叔只见证一件事：今天有人在现场复制并封存一张纸。两名普通人无法确认历史内容，却能确认当前动作发生。

杜岚同步记录露雨左手完成的每一步。周启明拒绝签，但没有关闭监控；他的在场知悉也被保留。

当前接收链成立后，玻璃里的手套没有消失，却一只只松开。它们仍属于过去，终于不必抓着今天的人维持未交接。

露雨把回执原件留在相框内，只抽出一层薄薄的复写纸。复写纸上没有患者姓名，只有送达路径：旧门诊楼——身份修复部——记录退回科。

第三站旁写着“拒收”。

身份修复部曾经接过材料，又把需要医疗继续处理的部分送回区域链。复写纸补上了旧复核页缺失的一段去向，却仍不能说明谁制造了最初病历，也不能证明露雨当时就在场。

露雨把复写纸封进袋里，右手彻底看不见。袖口空空垂下，她站稳时额头全是冷汗。

凉生没有问她疼不疼。这个问题太没用。他把自己的导诊胸牌翻过来，夹住档案袋封口，免得她单手系绳。

“胸牌借你三分钟，只当夹子。”他说，“不算接收人。”

“你越来越会写免责声明了。”

“被生活培养得比较全面。”

露雨笑了一下，很轻。她没有拒绝这点帮助。

两人的关系没有因为一次牺牲突然变成无条件信任。凉生仍不知道她完整身份，露雨也没把所有调查交给他。但他第一次按她设定的边界协助，而不是只把她当危险信息源。

取件结束后，露雨没有立刻离开。

她要求在场所有见证人逐项复述自己确认了什么。蒋师傅只确认有人复制封存材料；保安确认现场未发生原件强行带离；杜岚确认复写纸与旧转诊格式相符；凉生确认操作时段和露雨的当前声明。周启明只确认自己在场，没有给旧章现行效力。

没有人的一句话能独立把材料变成完整正史，组合起来却足以阻止它再次被写成从未发出。

露雨用左手在每份见证旁画出不同形状的缺口，避免系统将几份有限确认合并成“全体一致”。她做得很熟练，也因此暴露出一个新事实：她不是第一次处理被多方记录争抢的材料。

凉生看见了，没有追问她过去每一次都在哪里。当前最要紧的是她的右臂。

透明范围仍在往肩膀爬。露雨把右袖挽起，袖下没有血，也没有皮肤，只有背景穿过去后的轻微折射。她尝试拿起一支笔，笔直接穿过掌心落地。

“只是失去接触？”杜岚问。

“目前是。”

“感觉呢？”

“还有。”

这反而更糟。她能感觉手指握紧，却碰不到现实。杜岚没有把她当普通伤员包扎，只用布带固定空袖，避免移动时被门把或机械卷入。露雨接受护理前明确同意，并留下这是当前防护，不是入院或值班。

“你们两个说话越来越像。”周启明说。

“环境污染。”露雨回答。

杜岚系结时故意收紧一点，露雨吸了口气。两个女人没有因为都与凉生合作就自动亲近，也没有制造无意义敌意。她们都更关心材料会不会再次吞人。

杜岚问她：“如果这份复写纸能证明转诊送出，杜驰是不是就算到过观察床？”

“不能。只能证明同批材料走过这条路。”

“我知道。”杜岚低头看结，“还是想问。”

露雨停了停：“想问不等于越界。替他回答才是。”

这句回应让杜岚的手松了一点。她们之间建立的是对彼此边界的初步认可，不是朋友承诺。

凉生把胸牌从档案袋上取回时，系统试图记录他完成旧部门材料接收。露雨立刻用自己的声明顶回去，承担当前封存人；凉生则只承认胸牌作为物理夹具使用。两条记录都留下后，系统没有再把他们合成同一经手人。

露雨为此付出的右手代价也没有转给凉生。她脸色苍白，却仍坚持自己提着档案袋。凉生只替她开门，不碰袋身。

档案袋封好后，空相框玻璃里的蓝章转了半圈，露出章柄底部一行磨损小字：000不是患者编号。

是“无接收科室”的旧缩写。

周启明立刻说：“缩写不能证明患者000不存在。”

“也不能证明那张死亡病历属于具体死者。”凉生说。

相框里忽然传出微弱呼吸声。

不是老护士，也不是陌生女人。

导诊台上的患者000病历自己翻开，姓名栏仍写LS-0713，病床位置却变成了七张推床之间的空位。

杜岚挡在杜驰前面，露雨则把档案袋抱紧。凉生没有让任何人去迎。他先移开通道里的患者和急救箱，给未知事物留下不会撞到活人的空间。

那张一直没有人的第八张床，正从零层慢慢升上来。
